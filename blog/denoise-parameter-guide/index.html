denoise 參數完全指南：決定 AI 生圖品質的關鍵

denoise 是什麼？

denoise（去噪強度） 是 img2img 最重要的參數，它決定 AI要多大程度改變原圖。

簡單來說：

  • denoise 越低 → 越接近原圖
  • denoise 越高 → 改得越多

denoise 範圍速查

  • 0.3-0.4 — 微弱變化，幾乎原圖 ⭐⭐⭐ 結構保留高 → 細微修飾、顏色微調
  • 0.5 — ✅ 最佳平衡 ⭐⭐⭐ 結構保留高 → 風格化 + 保留結構
  • 0.6 — 風格更強 ⭐⭐ 結構保留中等 → 需要明顯風格變化
  • 0.7 — 強烈風格化 ⭐ 結構保留低 → 大幅改圖
  • 0.8-0.85 — 幾乎重繪 ⭐ 結構保留低 → Q版 / 手辦 / 完全重繪
  • 1.0 — 完全忽略原圖 → text-to-image

詳細實驗數據

denoise 0.3 — 微調模式

適用：想保留原圖 90% 以上，只加一點風格味

  • ✅ 人物特徵完全保留
  • ✅ 適合商業人像微修
  • ❌ 風格感不明顯

denoise 0.5 — 萬用設定（推薦）

適用：大多數情況的首選，平衡結構與風格

  • ✅ 結構保留與風格化的最佳平衡
  • ✅ 人物特徵可辨認
  • ✅ 風格轉換明顯

denoise 0.6 — 強風格模式

適用：需要明顯的風格轉換

  • ✅ 風格轉換明顯
  • ✅ 適合藝術風格化
  • ❌ 臉部特徵可能模糊

denoise 0.85 — Q版/手辦專用

適用：Q 版動漫、Pop Mart 手辦風格

  • ✅ Q 版效果最好
  • ✅ 手辦感強烈
  • ❌ 幾乎看不出原圖痕跡

denoise 1.0 — 純文字生圖

適用：text-to-image 模式，完全不看原圖

各風格推薦 denoise

  • 吉卜力：0.5 / cfg 7.0 / steps 20 — 最佳平衡
  • 新海誠：0.5 / cfg 7.0 / steps 20 — 保留結構
  • 水墨畫：0.5-0.6 / cfg 7.0 — ⚠️ 勿超過 0.7
  • 油畫：0.5-0.6 / cfg 7.0 — 人像最佳
  • 美式漫畫：0.6 / cfg 8.0 — 輪廓線更明顯
  • 水彩：0.5 / cfg 7.0 — 風景最佳
  • 像素風：0.6 / cfg 7.0 — 配合低解析度
  • Q 版動漫：0.85 / cfg 8.5 / steps 30 — 高 denoise 必備
  • 3D 手辦：0.85 / cfg 8.5 / steps 30 — 高 denoise 必備
  • 專業人像：0.5 / cfg 7.0 — 保留臉部特徵

cfg 參數補充

  • 5.0-6.0 — 較自由，模型有發揮空間 → 創意實驗
  • 7.0 — ✅ 平衡 → 推薦預設值
  • 8.0-9.0 — 高度忠於 prompt → 精準控制
  • >9.0 — ⚠️ 可能產生色偏或異常

實戰策略

步驟 1：從 0.5 開始

先用 denoise 0.5 + cfg 7.0 + steps 20，這是大多數情況的萬用設定。

步驟 2：根據結果調整

  • 結構保留太多 → 提高 denoise（+0.1）
  • 風格不夠強 → 提高 cfg（+0.5）
  • 畫面變形 → 降低 denoise（-0.1）
  • 細節不夠 → 增加 steps（+10）

步驟 3：記錄最佳設定

每張好圖都記錄用的參數，建立自己的參數庫。

常見問題

Q: denoise 設 0.85 但人物消失了？
→ 這是正常的，0.85 就是大幅改變。若需保留人物特徵，改用 0.5-0.6。

Q: 為什麼同樣 denoise 結果不同？
→ seed 不同、prompt 不同、原圖內容不同都會影響結果。

Q: steps 設多高比較好？
→ 20 是 Sweet Spot，30 以上邊際效益遞減，但耗時增加。

Q: denoise 和 cfg 哪個更重要？
→ denoise 決定「改變多少」，cfg 決定「改變方向」。兩者搭配使用。
