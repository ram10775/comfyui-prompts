img2img 是什麼？

img2img（圖片生圖片）的作法很簡單：

  1. 輸入一張照片 — 你的底圖
  2. 寫一段描述 — 你想轉換成什麼風格
  3. 調整 denoise — 決定改變程度
  4. 輸出結果 — AI 根據你的設定重新繪製

準備工作流

ComfyUI 基本 img2img 流程

LoadImage → VAE Encode → KSampler → VAE Decode → SaveImage
                    ↑
           CLIPTextEncode (你的 prompt)
                    ↑
           CLIPLoader (Qwen 模型)

必要節點

  • LoadImage — 載入你的照片
  • VAE Encode — 將圖片轉換為 latent 空間
  • KSampler — 核心採樣器，負責生圖
  • VAE Decode — 將 latent 還原為圖片
  • SaveImage — 儲存輸出

實戰技巧

1. 照片選擇

適合的：光線均勻、構圖清晰、主體明顯

不適合的：太暗或過曝、模糊不清、構圖雜亂

2. 預處理建議

  1. 裁剪到合理比例（1:1 或 16:9）
  2. 亮度/對比度微調
  3. 去除不需要的雜物

3. 結果不滿意時

  • 人物變形 → 降低 denoise（-0.1）
  • 風格不明顯 → 提高 cfg（+0.5）
  • 顏色不對 → 在 prompt 中加入色調描述
  • 細節模糊 → 增加 steps（+10）
  • 人物消失 → 大幅降低 denoise（0.4-0.5）
