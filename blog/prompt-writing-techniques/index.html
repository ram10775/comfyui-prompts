AI 繪圖 Prompt 撰寫技巧：從新手到大師的 7 個秘訣

技巧2026-05-11⏱ 12 分鐘 prompt技巧進階AI繪圖

為什麼 Prompt 這麼重要？

好的 Prompt 可以：

  • ✅ 精準表達你想要的畫面
  • ✅ 減少試錯次數，省時間
  • ✅ 提升圖片品質和一致性
  • ✅ 讓 AI 更理解你的意圖

「垃圾進，垃圾出」— 這句老話在 AI 繪圖上完全適用。

秘訣 1：明確的主題描述

不要寫：一隻貓

要寫：一隻橘色的虎斑貓，坐在窗台上，陽光灑在身上，閉著眼睛在打盹

公式：主題 + 特徵 + 動作 + 環境

秘訣 2：具體的風格定義

不要寫：好看的風格

要寫：

  • 吉卜力工作室風格，宮崎駿水彩畫風
  • 新海誠風格，高飽和度，夢幻光線
  • 寫實攝影，50mm 鏡頭，淺景深

秘訣 3：光線控制

光線決定畫面氛圍，善用這些關鍵詞：

  • 自然光：柔和陽光、晨光、夕陽餘暉
  • 人工光：霓虹燈、燭光、舞台燈光
  • 特殊光：逆光、側光、頂光、丁達爾效應
柔和的自然光從窗戶照進，營造溫暖的氛圍
逆光勾勒輪廓，髮絲被陽光染成金色

秘訣 4：構圖技巧

加入構詞描述，讓畫面更有層次：

  • 近景特寫，淺景深，背景模糊
  • 全景，廣角鏡頭，前景中景背景分明
  • 俯拍，鳥瞰視角
  • 低角度仰拍，突出主體

秘訣 5：色彩與情緒

色彩影響情緒表達：

  • 溫暖：橙黃色調、夕陽光、暖色調
  • 清冷：藍灰色調、月光、冷色調
  • 夢幻：粉紫色調、柔光、高飽和度
  • 復古：棕褐色調、顆粒感、褪色效果

秘訣 6：細節描述

增加細節讓畫面更生動：

  • 花瓣上有水珠，晨露閃閃發光
  • 木頭紋理清晰可見，有使用痕跡
  • 衣服皺摺自然，布料質感柔軟

秘訣 7：負面提示（Negative Prompt）

告訴 AI 不要產生什麼：

模糊，低品質，變形，多餘的手指，
奇怪的比例，文字，水印，签名

完整範例

吉卜力工作室風格，宮崎駿水彩畫風，
少女坐在老樹下看書，
柔和的自然光從樹葉間灑下，
翠綠的草地，遠處有白色的雲朵，
近景特寫，淺景深，
溫暖的橙黃色調，
花瓣上有水珠，晨露閃閃發光

練習方法

  1. 模仿：從好的範例開始，逐句修改
  2. 拆解：把好的 Prompt 拆成元素，理解每個部分
  3. 組合：把不同元素組合，創造新 Prompt
  4. 記錄：建立自己的 Prompt 資料庫
