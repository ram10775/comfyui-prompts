AI 繪圖 Prompt 撰寫技巧：從新手到大師的 7 個秘訣

技巧2026-05-11⏱ 12 分鐘prompt技巧進階教學

Prompt 是什麼？

Prompt（提示詞）是告訴 AI「你要畫什麼」的指令。寫得好不好，直接決定出圖品質。一個好的 Prompt 就像給攝影師的拍攝簡報 — 越具體，結果越好。

秘訣 1：主題先行（Subject First）

Prompt 的第一個元素永遠是你的主體，這會讓 AI 把最多注意力放在這裡。

✅ 好的寫法

一位穿著白色洋裝的女性，站在海邊，面向夕陽

❌ 不好的寫法

夕陽海景，白色洋裝，女性站在沙灘上

原因：AI 對前面詞彙的權重較高，把主體放前面確保它成為畫面焦點。

秘訣 2：風格明確（Style Definition）

不要只寫「好看」或「精美」，要用具體的藝術風格來描述。

風格關鍵字

  • 吉卜力：吉卜力工作室風格，宮崎駿水彩畫風，手繪動畫質感
  • 新海誠：新海誠風格，高飽和度藍天白雲，電影感光影
  • 水墨畫：中國傳統水墨畫風格，潑墨渲染，留白藝術
  • 油畫：歐洲古典油畫風格，厚塗筆觸，巴洛克光線
  • 美式漫畫：美式漫畫風格，粗黑輪廓線，網點陰影
  • 水彩：水彩畫風格，輕透色彩疊加，自然水痕
  • 像素風：16-bit 像素遊戲風格，復古遊戲角色
  • Q 版：Q 版動漫風格，大眼睛，簡化特徵

秘訣 3：光線描述（Lighting）

光線是照片與插畫的靈魂。加入光線描述會讓畫面更有層次。

# 自然光
柔和的陽光 / 黃金時段光線 / 清晨柔光 / 逆光剪影
# 人造光
棚燈照明 / 霓虹燈光 / 燭光 / 聚光燈
# 特殊光線
電影感光影 / 戲劇性光影 / 逆光 / 光暈效果 / 光斑

秘訣 4：構圖與視角（Composition）

# 景別
特寫（臉部）/ 半身 / 全身 / 遠景 / 廣角
# 視角
俯視 / 仰視 / 平視 / 航拍視角 / 第一人稱視角
# 構圖
中心構圖 / 三分法 / 對稱構圖 / 黃金螺旋

秘訣 5：細節堆疊（Detail Stacking）

[主體] + [服裝/外觀] + [動作] + [場景] + [光線] + [風格] + [品質詞]

完整範例

一位穿著和服的日本女性，黑色長髮綁成髮髷，
手執油紙傘，站在京都嵐山竹林小徑，晨霧繚繞，
柔和的晨光穿透竹葉，日本浮世繪風格，木版畫質感，高清 4K

秘訣 6：負面提示詞（Negative Prompt）

低品質，模糊，失真，變形，多手指，額外肢體，
水印，文字，邊框，裁切，過曝，欠曝，醜陋，畸形

秘訣 7：權重控制（Weight Control）

cfg 7.0 — 平衡（推薦）| cfg 8.5 — 更忠於 prompt | cfg 5.0 — AI 更多發揮空間

實戰範例

範例 1：吉卜力風格人像

吉卜力工作室風格，宮崎駿水彩畫風，
一位穿著白色連衣裙的少女，棕色短髮，坐在翠綠的草地上，
頭頂是大朵白雲的藍天，柔和的陽光，溫暖寧靜的氛圍，手繪動畫質感

範例 2：商業產品照

專業產品攝影，純白背景，精緻的機械手錶，銀色錶殼，藍面錶盤，
柔和的側面光線，無反光，4K 高清，簡潔構圖，商品目錄質感

範例 3：美食攝影

美食攝影，自然光從側面照射，一碗拉麵，叉燒、半熟蛋、海苔、蔥花，
木質桌面背景，蒸汽升起，微距鏡頭，淺景深，暖色調，餐廳宣傳照質感

快速修改策略

  • 風格不滿意 → 替換風格關鍵字
  • 主體偏移 → 把主體移到 prompt 最前面
  • 畫面太暗 → 加入「明亮的陽光」、「暖色調」
  • 細節不足 → 增加 cfg 到 8.0-8.5
  • 變形嚴重 → 加入負面提示詞

記住：Prompt 沒有標準答案，多嘗試、多比較，找到屬於你的寫法。
